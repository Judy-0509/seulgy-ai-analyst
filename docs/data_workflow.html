Step 6 주제 선정
기준
Cross-Layer Corroboration Rules
Criterion 2 — Multi-Source Signal
30일 내 2개 이상 독립 소스 → 동일 시장 현상 보도
"동일 현상"은 키워드 일치가 아닌 의미적 동일성으로 판단.

강력 신호 조합 (Cross-Layer):
• Layer C (Yole 부품) + Layer A (Counterpoint 출하) → 기술 양산 양면 확정
• Layer E (DigiTimes leak) + Layer A (Tracker 확인) → 루머 데이터 검증
• Layer F (CCS 채널) + Layer A (IDC 출하) → sell-through 일치 신호

※ 반대 방향도 동일 현상: A가 성장 / B가 하락 → 시장 갈등 시그널
Criterion 3 — Emerging Signal
단일 소스, 높은 참신성 → Curiosity Pick
7일 단기 창에서 suggest_smartphone_emerging.py 별도 실행.

패턴 유형:
(a) New entrant — 신규 플레이어 등장
(b) Contrarian — 지배적 내러티브와 정반대 신호
(c) Tech fact — 기술 파라미터 수치 확정
(d) Setback — 예상치 못한 시장 후퇴

※ 기존 보고서 주제와 중복이면 Criterion 3 제외

• Layer A + Layer C (arXiv) → 기술 구현 + 학문적 근거
